Steps to compile the programs:
1. A file named data.txt will need to be in the current directory.
(if it is not in the directory you can pass in the file path as the only
arguement to the program when running the executable.)
2. on flip run the following commands

g++ -O3 -o insertsort insertsort.cpp

g++ -O3 -o mergesort mergesort.cpp

3. Then run the executables to run the program.
